## Data Sources

### HUD System Performance Measures (SPM) Data
- **Source**: HUD Exchange
- **URL**: https://www.hudexchange.info/resource/5691/system-performance-measures-data-since-fy-2015/
- **Description**: System Performance Measures data from FY 2015 onwards

### U.S. Census Bureau Population Estimates

#### 2020s County Population Estimates (`co-est2024-pop`)
- **Source**: U.S. Census Bureau
- **URL**: https://www.census.gov/data/tables/time-series/demo/popest/2020s-counties-total.html
- **Instructions**: Select "United States" from available tables

#### 2010s County Population Estimates (`co-est2019-annres`)
- **Source**: U.S. Census Bureau
- **URL**: https://www.census.gov/data/tables/time-series/demo/popest/2010s-counties-total.html
- **Instructions**: Select "United States" from available tables

### Bureau of Labor Statistics (BLS) Unemployment Data
- **Source**: BLS Local Area Unemployment Statistics
- **URL**: https://www.bls.gov/lau/tables.htm#cntyaa
- **Data Files**: Labor force data by county, annual averages (XLSX format)
  - 2024 annual averages
  - 2023 annual averages
  - 2022 annual averages
  - 2021 annual averages
  - 2020 annual averages
  - 2019 annual averages
  - 2018 annual averages
  - 2017 annual averages
  - 2016 annual averages

### COVID-19 Data
- **Source**: New York Times COVID-19 Database
- **URL**: https://github.com/nytimes/covid-19-data
- **Data Files**:
  - `us-counties-2020.csv`
  - `us-counties-2021.csv`
  - `us-counties-2022.csv`
  - `us-counties-2023.csv`

### Geographic Boundary Files

#### Continuum of Care (CoC) Shapefiles
- **Source**: HUD Exchange GIS Tools
- **URL**: https://www.hudexchange.info/programs/coc/gis-tools/?filter_ToolType=&filter_Year=&filter_State=&program=CoC&group=GIS
- **Instructions**: Select "2023 CoC GIS Geodatabase - All CoCs"

#### County Shapefiles
- **Source**: U.S. Census Bureau TIGER/Line Shapefiles
- **URL**: https://www2.census.gov/geo/tiger/TIGER2023/COUNTY/
- **File**: `tl_2023_us_county.zip`
